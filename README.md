# Ground Control
* Ground Control is a React Based front end Dashboard that is meant to provide quick pre-boarding information to pilots and flight attendants.
The dashboard displays Flight number, route, crew status, and boarding status. 

## Technologies
* React
* Javascript
* Vite
* Tailwind CSS
